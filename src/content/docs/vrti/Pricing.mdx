VRTI is free to try in its evaluation mode for as long as you'd like.
If you find it useful and would like to continue using it, you can purchase a license for a one-time fee of $8.

The evaluation mode contains most of VRTI's basic features and is intended for those who wish to try out VRTI before purchasing a license.
If you wish to continue using VRTI after evaluating it, you are asked to purchase a license.
